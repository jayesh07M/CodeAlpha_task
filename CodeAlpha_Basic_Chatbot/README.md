## Task Name
Basic Chatbot

## Description
This project is a simple rule-based chatbot created using Python.  
The chatbot gives predefined responses to predefined user inputs such as  
"hello", "how are you", and "bye".

## Functionality
- Takes input from the user
- Matches the input using if-elif conditions
- Displays predefined responses
- Ends the chat when the user types "bye"

## Concepts Used
- Python Functions
- if-elif-else statements
- while loop
- input and output

## How to Run
1. Install Python on your system
2. Open terminal or command prompt
3. Run the program using:
   python chatbot.py

## Example
Input:
hello  
how are you  
bye  

Output:
Hi! Nice to meet you  
I'm fine, thanks for asking  
Goodbye! Have a great day
